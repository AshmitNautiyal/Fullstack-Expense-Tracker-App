# Fullstack Expense Tracker

A full-stack expense tracking web application built using the PERN stack that allows users to securely manage and monitor their expenses with a modern, responsive interface.

Visit here : https://my-finance-w7it.onrender.com

## Features

1.User authentication & authorization (secure access)

2.Add, edit, and delete expenses

3.Expense analytics with charts

4.Real-time total expense calculation

5.Responsive UI with dark/light mode

6.RESTful API for backend operations

7.Persistent data storage with PostgreSQL


## Tech Stack

Frontend: React.js, Tailwind CSS

Backend: Node.js, Express.js

Database: PostgreSQL

API: RESTful APIs

## Installation & Setup
1️⃣ Clone the repository

git clone[ https://github.com/your-username/expense-tracker.git](https://github.com/AshmitNautiyal/Fullstack-Expense-Tracker-App.git)
cd expense-tracker

2️⃣ Backend Setup

cd server

npm install

npm run dev

3️⃣ Frontend Setup

cd client

npm install

npm start

4️⃣ Environment Variables

Create a .env file in the server folder:

PORT=5000

DATABASE_URL=your_postgresql_url
